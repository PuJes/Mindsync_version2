# 📦 AI 本地知识库 - 本地与手机使用指南

本指南将教您如何在电脑本地运行本应用，并使其在您的手机上可用（像 App 一样安装）。

## 🖥️ 1. 电脑端启动 (Server)

确保您已经在项目目录下打开终端。

1. **安装依赖** (如果是首次运行):
   ```bash
   npm install
   ```
2. **启动服务**:
   ```bash
   npm run dev
   ```
   终端会显示类似如下的输出：
   ```
     ➜  Local:   http://localhost:3000/
     ➜  Network: http://192.168.1.5:3000/  <-- 记下这个 Network 地址
   ```

## 📱 2. 手机端访问 (Mobile)

**前提条件**: 您的手机和电脑必须连接到 **同一个 Wi-Fi** 网络。

1. **打开手机浏览器** (推荐 Chrome 或 Safari)。
2. **输入地址**: 输入上一步中看到的 `Network` 地址 (例如 `http://192.168.1.5:3000`)。
3. **访问成功**: 您应该能看到知识库界面。

## 📲 3. 安装到手机桌面 (PWA)

为了获得类似原生 App 的体验（全屏运行、桌面图标），您可以将其“安装”到手机上。

### iOS (Safari)
1. 在 Safari 中打开上述网址。
2. 点击底部的 **分享按钮** (方框带箭头)。
3. 向下滚动，点击 **“添加到主屏幕”** (Add to Home Screen)。
4. 点击右上角的 **“添加”**。
5. 现在桌面上会出现“AI知识库”的图标，点击即可全屏运行。

### Android (Chrome)
1. 在 Chrome 中打开上述网址。
2. 可能会自动弹出 **“添加到主屏幕”** 的提示，点击即可。
3. 如果没有弹出，点击右上角的 **三个点菜单**。
4. 选择 **“安装应用”** 或 **“添加到主屏幕”**。

## ⚠️ 注意事项

1. **数据存储**:
   - 本应用目前使用浏览器本地存储 (IndexedDB/LocalStorage)。
   - **电脑上**添加的文件/数据保存在**电脑浏览器**里。
   - **手机上**添加的文件/数据保存在**手机**里。
   - **它们目前不互通！** (手机只是访问了电脑提供的代码页面，数据是隔离的)。
   
2. **防火墙**:
   - 如果手机打不开页面，请检查电脑的 **防火墙设置**，确保允许 Node.js/Vite 通过防火墙。

3. **API Key**:
   - 手机初次访问时，需要重新在“设置”中输入您的 DeepSeek 或 Gemini API Key。
